# Java Programming Tasks

This repository contains a collection of Java programming tasks designed to enhance understanding of core Java concepts and practice coding skills. Each task is implemented as a standalone Java application.

## Tasks

### Task 1: Number Game
A simple number guessing game with features such as limited attempts, multiple rounds, and scoring.

### Task 2: Student Grade Calculator
A calculator that computes total marks, average percentage, and assigns grades based on marks obtained in each subject.

### Task 3: ATM Interface
An ATM simulation that allows users to withdraw, deposit, and check their account balance with input validation and transaction messages.

### Task 4: Currency Converter
A currency converter that allows users to convert amounts between various currencies using simulated exchange rates.

## How to Run
1. Clone the repository.
2. Open the project in your preferred IDE (e.g., NetBeans, IntelliJ IDEA).
3. Navigate to the respective task directory.
4. Compile and run the main class for each task.

## Requirements
- JDK 8 or higher
- Maven or Ant (depending on the build tool you choose)
